# 사진 자동 분류 프로그램

단체 사진을 아이별로 자동으로 나눠주는 프로그램입니다.

---

## 시작하는 방법

`실행.bat` 파일을 **더블클릭**하세요.

- 처음 실행할 때는 필요한 프로그램을 자동으로 설치합니다 (시간이 좀 걸릴 수 있어요)
- 설치가 끝나면 인터넷 브라우저가 자동으로 열립니다
- 두 번째 실행부터는 바로 열립니다

---

## 사용 방법

### 1. 아이 등록

1. 아이 이름을 입력합니다
2. 그 아이의 사진을 1장 이상 올립니다 (얼굴이 잘 보이는 사진일수록 좋아요)
3. **등록** 버튼을 누릅니다
4. 모든 아이를 같은 방법으로 등록합니다

> 사진은 나중에 **+ 추가** 버튼으로 더 올릴 수 있어요.
> 아이당 3~5장을 다양한 각도로 올리면 더 정확합니다.

### 2. 분류할 사진 폴더 지정

분류하고 싶은 사진들이 들어있는 폴더 경로를 입력합니다.

예시: `C:\Users\선생님\Desktop\졸업사진`

### 3. 민감도 설정

슬라이더로 인식 민감도를 조절할 수 있어요.

- **0.25~0.30**: 엄격 (확실한 것만 분류, 놓치는 사진 많음)
- **0.33~0.37**: 권장 (적당한 균형)
- **0.38~0.40**: 느슨 (인식률 높음, 오분류 가능성 있음)

### 4. 분류 시작

**분류 시작** 버튼을 누르면 자동으로 분류가 시작됩니다.
진행 상황을 실시간으로 확인할 수 있어요.

---

## 결과 예시

완료되면 원래 사진 폴더 안에 아이 이름으로 폴더가 생깁니다.

```
졸업사진/
├── 홍길동/      ← 홍길동이 찍힌 사진들
├── 김철수/      ← 김철수가 찍힌 사진들
└── _미분류/     ← 누구인지 알 수 없는 사진들
```

---

## 기타 기능

- **+ 추가**: 등록된 아이에 기준사진을 추가로 올릴 수 있어요
- **삭제**: 아이를 개별 삭제할 수 있어요
- **전체 초기화**: 등록된 아이를 한 번에 모두 삭제합니다

---

## 프로그램 종료

브라우저 탭을 닫거나 검은 창(CMD)을 닫으면 프로그램이 종료됩니다.
(둘 중 하나만 닫아도 나머지가 자동으로 같이 종료돼요)
